# GIMP 使用文档

## 1. 概述

GIMP（GNU Image Manipulation Program）是一款免费、开源的光栅图像编辑软件，用于图像编辑、修饰、合成和创作。它是 Adobe Photoshop 的强大替代品，支持多种操作系统。

### 1.1 特点
- **完全免费**：开源软件，无需付费
- **跨平台**：支持 Windows、macOS、Linux
- **专业功能**：媲美商业软件的图像处理能力
- **高度可定制**：支持插件、脚本、主题
- **格式丰富**：支持几乎所有图像格式

### 1.2 系统要求
- **Windows**：Windows 7 或更高版本
- **macOS**：macOS 10.12 或更高版本
- **Linux**：大多数主流发行版
- **内存**：建议 4GB 以上
- **硬盘**：至少 1GB 可用空间

### 1.3 下载安装
- 官网：https://www.gimp.org/
- 支持安装版和便携版

## 3. 基本操作

### 3.1 新建图像

```
文件 → 新建
快捷键：Ctrl + N
```

**设置选项：**
- 宽度/高度：图像尺寸
- 分辨率：像素/英寸
- 颜色模式：RGB、灰度、索引
- 填充：背景色、前景色、透明

### 3.2 打开文件

```
文件 → 打开
快捷键：Ctrl + O
```

**支持格式：**
- XCF - GIMP 原生格式
- PSD - Photoshop 格式
- PNG、JPEG、GIF、BMP、TIFF
- SVG、PDF、EPS
- RAW 格式

### 3.3 保存和导出

**保存为 XCF：**
```
文件 → 保存
快捷键：Ctrl + S
```

**导出为其他格式：**
```
文件 → 导出为
快捷键：Ctrl + Shift + E
```

**常见导出格式：**
- PNG - 透明背景、无损压缩
- JPEG - 照片、有损压缩
- GIF - 动画、256 色
- TIFF - 印刷、高质量
- PDF - 文档、打印

### 3.4 撤销和重做

```
撤销：Ctrl + Z
重做：Ctrl + Shift + Z
```

### 3.5 缩放和平移

**缩放：**
```
放大：Ctrl + + 或 Ctrl + 滚轮
缩小：Ctrl + - 或 Ctrl + 滚轮
适合窗口：Ctrl + Shift + E
100% 视图：Ctrl + 1
```

**平移：**
- 按住空格键拖动
- 使用滚动条
- 中键拖动

## 13. 批量处理

### 13.1 批量转换格式

使用命令行：
```bash
gimp -i -b '(gimp-file-load RUN-NONINTERACTIVE "input.jpg" "input.jpg")' \
     -b '(gimp-file-save RUN-NONINTERACTIVE 1 0 "output.png" "output.png")' \
     -b '(gimp-quit 0)'
```

### 13.2 批量调整大小

使用 Script-Fu 脚本：
```scheme
(let* (
  (filelist (cadr (file-glob "*.jpg" 1)))
  (filename (car filelist))
  )
  (while (not (null? filename))
    (let* (
      (image (car (gimp-file-load RUN-NONINTERACTIVE filename filename)))
      )
      (gimp-image-scale image 800 600)
      (gimp-file-save RUN-NONINTERACTIVE image (car (gimp-image-get-active-drawable image)) filename filename)
      (gimp-image-delete image)
    )
    (set! filelist (cdr filelist))
    (set! filename (car filelist))
  )
)
```

- PSD - Photoshop 格式
- PNG、JPEG、GIF、BMP、TIFF

### 16.2 导出格式

**常用导出格式：**

| 格式 | 用途 | 特点 |
|------|------|------|

| GIF | 动画 | 256 色、支持动画 |

| WebP | 网页 | 高效压缩 |
| SVG | 矢量图形 | 可缩放 |

### 16.3 导出设置

**PNG 导出：**
- 压缩级别：0-9
- 交错：逐行加载
- 保存背景色：透明度处理

**JPEG 导出：**
- 质量：0-100
- 优化：文件大小优化
- 渐进式：逐行加载

## 18. 常见问题

### 18.1 性能优化

**问题：** GIMP 运行缓慢

**解决方案：**
- 增加内存分配：编辑 → 首选项 → 环境
- 减少撤销次数
- 关闭不必要的面板
- 使用较小的画笔
- 降低图像分辨率进行编辑

### 18.2 文件格式问题

**问题：** 无法打开特定格式

**解决方案：**
- 安装格式插件
- 转换格式后再打开
- 使用在线转换工具

### 18.3 颜色管理

**问题：** 颜色显示不准确

**解决方案：**
- 配置颜色管理：编辑 → 首选项 → 颜色管理
- 使用 ICC 配置文件
- 校准显示器

### 18.4 打印问题

**问题：** 打印效果与屏幕不符

**解决方案：**
- 使用 CMYK 颜色模式
- 设置正确的分辨率（300 DPI）
- 导出为 TIFF 或 PDF
- 使用打印服务提供的配置文件

### 18.5 插件问题

**问题：** 插件无法加载

**解决方案：**
- 检查插件路径
- 确认插件版本兼容
- 查看错误日志
- 重新安装插件
