# Inkscape 使用文档

## 1. 概述

Inkscape 是一款免费、开源的矢量图形编辑器，用于创建和编辑 SVG（可缩放矢量图形）文件。它提供了专业级的绘图工具，适用于插图、图标、徽标、图表、地图等设计工作。

### 1.1 特点
- **完全免费**：开源软件，无需付费
- **跨平台**：支持 Windows、macOS、Linux
- **专业功能**：媲美商业软件的绘图能力
- **SVG 原生**：基于 W3C SVG 标准
- **可扩展**：支持插件和脚本扩展

### 1.2 系统要求
- **Windows**：Windows 7 或更高版本
- **macOS**：macOS 10.13 或更高版本
- **Linux**：大多数主流发行版

### 1.3 下载安装
- 官网：https://inkscape.org/
- 支持安装版和便携版

## 3. 基本操作

### 3.1 新建文档

```
文件 → 新建
快捷键：Ctrl + N
```

### 3.2 保存文件

```
保存：Ctrl + S
另存为：Ctrl + Shift + S
保存副本：Ctrl + Shift + Alt + S
```

**支持格式：**
- `.svg` - SVG 格式（默认）
- `.svgz` - 压缩 SVG
- `.pdf` - PDF 格式
- `.eps` - EPS 格式
- `.png` - PNG 位图
- `.jpg` - JPEG 位图

### 3.3 撤销和重做

```
撤销：Ctrl + Z
重做：Ctrl + Shift + Z
```

### 3.4 缩放操作

```
放大：Ctrl + + 或 Ctrl + 滚轮
缩小：Ctrl + - 或 Ctrl + 滚轮
适合页面：Ctrl + Shift + E
适合选区：Ctrl + Shift + R
100% 缩放：Ctrl + 1
```

## 4. 绘图工具详解

### 4.1 矩形工具 (R)

**绘制矩形：**
1. 选择矩形工具（R）
2. 拖动鼠标绘制
3. 按住 Shift 从中心绘制
4. 按住 Ctrl 绘制正方形

**圆角矩形：**
- 拖动矩形右上角的圆角控制点
- 或在工具选项栏设置 rx、ry 值

**属性设置：**
- 位置：X、Y 坐标
- 尺寸：宽度、高度
- 圆角：rx、ry

### 4.2 椭圆工具 (E)

**绘制椭圆：**
1. 选择椭圆工具（E）
2. 拖动鼠标绘制
3. 按住 Shift 从中心绘制
4. 按住 Ctrl 绘制正圆

**弧形和扇形：**
- 拖动椭圆上的弧形控制点
- 在工具选项栏选择模式：
  - 椭圆
  - 弧形
  - 扇形

### 4.3 多边形/星形工具 (*)

**绘制多边形：**
1. 选择多边形工具（*）
2. 在工具选项栏设置边数
3. 拖动鼠标绘制

**绘制星形：**
- 在工具选项栏切换到星形模式
- 设置角数和外径比例

**参数设置：**
- 边数/角数：3-1024
- 外径比例：星形的尖锐程度
- 圆角：边角圆滑程度
- 随机：随机偏移

### 4.4 铅笔工具 (P)

**自由绘制：**
1. 选择铅笔工具（P）
2. 按住鼠标拖动绘制
3. 释放鼠标完成

**选项设置：**
- 形状：三角形、椭圆
- 宽度：笔触宽度
- 模式：
  - 自由绘制
  - 循环绘制

### 4.5 钢笔工具 (B)

**绘制贝塞尔曲线：**
1. 选择钢笔工具（B）
2. 点击创建直线点
3. 拖动创建曲线点
4. 双击或按 Enter 完成

**快捷操作：**
- 点击：创建尖角点
- 拖动：创建平滑点
- 双击：结束路径
- Esc：取消绘制

**节点类型：**
- 尖角点：无控制柄
- 平滑点：对称控制柄
- 对称点：等长控制柄

### 4.6 画笔工具 (V)

**书法笔触：**
1. 选择画笔工具（V）
2. 设置笔触参数
3. 绘制

**参数设置：**
- 宽度：笔触宽度
- 角度：笔触角度
- 稳定器：平滑绘制

### 4.7 文本工具 (T)

**创建文本：**
1. 选择文本工具（T）
2. 点击创建点文本
3. 拖动创建区域文本
4. 输入文字

**文本属性：**
- 字体：字体选择
- 大小：字号
- 样式：粗体、斜体
- 对齐：左对齐、居中、右对齐
- 间距：行距、字距

**文本转换为路径：**
```
路径 → 对象转路径
快捷键：Ctrl + Shift + C
```

### 4.8 渐变工具 (G)

**创建渐变：**
1. 选择对象
2. 选择渐变工具（G）
3. 拖动创建渐变

**渐变类型：**
- 线性渐变
- 径向渐变

**编辑渐变：**
- 拖动端点调整方向
- 点击渐变条添加色标
- 双击色标修改颜色

### 4.9 滴管工具 (D)

**颜色取样：**
1. 选择滴管工具（D）
2. 点击要取样的颜色
3. 颜色会应用到当前填充或描边

**快捷操作：**
- 点击：取样填充颜色
- Shift + 点击：取样描边颜色

## 5. 对象操作

### 5.1 选择对象

**选择工具 (S)：**
- 点击：选择单个对象
- Shift + 点击：添加/移除选择
- 拖动：框选对象
- Ctrl + A：全选

**选择所有：**
```
编辑 → 全选
快捷键：Ctrl + A
```

### 5.2 移动对象

**移动方式：**
- 拖动：鼠标移动
- 方向键：微调位置（1px）
- Shift + 方向键：大步移动（10px）
- 在属性栏输入精确坐标

### 5.3 缩放对象

**缩放操作：**
- 拖动控制点缩放
- 按住 Ctrl 保持比例
- 在属性栏输入精确尺寸

### 5.4 旋转对象

**旋转操作：**
- 双击对象显示旋转控制点
- 拖动角控制点旋转
- 按住 Ctrl 限制 15° 步进
- 在属性栏输入精确角度

### 5.5 倾斜对象

**倾斜操作：**
- 双击对象显示倾斜控制点
- 拖动边控制点倾斜

### 5.6 复制和粘贴

```
复制：Ctrl + C
粘贴：Ctrl + V
原位粘贴：Ctrl + Alt + V
复制对象：Ctrl + D
```

### 5.7 分组和取消分组

```
分组：Ctrl + G
取消分组：Ctrl + Shift + G
```

### 5.8 排列顺序

```
移到最前：Home
移到最后：End
上移一层：Page Up
下移一层：Page Down
```

## 14. 导入和导出

```
文件 → 导入
快捷键：Ctrl + I
```

**支持格式：**
- SVG、SVGZ
- AI、EPS、PDF
- PNG、JPEG、GIF、BMP
- DXF、WMF、EMF

### 14.2 导出文件

**导出为 PNG：**
```
文件 → 导出 PNG 图像
快捷键：Ctrl + Shift + E
```

**导出设置：**
- 分辨率：DPI
- 尺寸：宽度、高度
- 区域：页面/选择/自定义
- 背景：透明/白色

**导出为 PDF：**
```
文件 → 另存为 → PDF
```

**导出为其他格式：**
```
文件 → 另存为 → 选择格式
```

### 14.3 批量导出

使用命令行工具：
```bash
inkscape --export-type=png --export-dpi=300 *.svg
```

## 15. 插件和扩展

### 15.1 安装插件

**手动安装：**
1. 下载插件文件
2. 放入扩展目录：
   - Windows：`%APPDATA%\inkscape\extensions\`
   - macOS：`~/.config/inkscape/extensions/`
   - Linux：`~/.config/inkscape/extensions/`
3. 重启 Inkscape

### 15.2 常用插件

**生成类：**
- 条形码生成器
- 二维码生成器
- 日历生成器
- 图表生成器

**路径类：**
- 路径效果库
- 形状生成器
- 图案生成器

**工具类：**
- 批量处理工具
- 颜色管理工具
- 导出工具

### 15.3 自定义扩展

**Python 脚本：**
```python
#!/usr/bin/env python
import inkex

class MyEffect(inkex.Effect):
    def __init__(self):
        inkex.Effect.__init__(self)
    
    def effect(self):
        # 处理逻辑
        pass

if __name__ == '__main__':
    e = MyEffect()
    e.affect()
```

## 18. 常见问题

### 18.1 性能优化

**问题：** 文件过大，操作卡顿

**解决方案：**
- 简化路径（Ctrl + L）
- 减少节点数量
- 使用符号和克隆
- 分离复杂效果
- 降低显示质量

### 18.2 兼容性问题

**问题：** SVG 在其他软件中显示异常

**解决方案：**
- 使用标准 SVG 元素
- 避免使用滤镜
- 转换文本为路径
- 保存为 SVG 1.1

### 18.3 导出问题

**问题：** 导出 PNG 质量差

**解决方案：**
- 提高 DPI 设置
- 使用抗锯齿
- 检查背景设置
- 选择正确的导出区域

### 18.4 打印问题

**问题：** 打印效果与屏幕不符

**解决方案：**
- 使用 CMYK 颜色模式
- 设置正确的出血
- 导出为 PDF 打印
- 检查分辨率设置
